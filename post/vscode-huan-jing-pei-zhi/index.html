2022-04-11

vscode锁无法获取

vscode也存在一定的坑，例如说：
Server installation process already in progress - waiting and retrying
Acquiring lock on ~/.vscode-server/bin/
一般的解决方法就是删除bin目录，然后重新下载，这是因为不同电脑的vscode版本可能不同，如果不同版本的vscode同样来连接服务器，就会存在一些锁的问题。
